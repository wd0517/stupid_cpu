# Stupid CPU

> 以无脑的方式消耗 CPU, 将 CPU 使用率控制在 30%
> 注意: 至少占用1个 CPU 1% 的资源

## Install

```
pip install git+https://github.com/wd0517/stupid_cpu.git
```

## Update

```
pip install -U git+https://github.com/wd0517/stupid_cpu.git
```

## Run

```
run_stupid_cpu
```

## Platform

- Linux + cgroup

## Requirements

- Python3
- cgroups
- psutil
